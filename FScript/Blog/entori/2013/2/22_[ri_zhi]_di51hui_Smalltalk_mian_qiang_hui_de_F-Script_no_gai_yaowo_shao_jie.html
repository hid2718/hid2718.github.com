［日誌］第51回 Smalltalk勉強会 でプレゼンを行った： F-Script の概要紹介＆デモ

 

今日は、Smalltalk-users-jp の梅澤さんが東京で毎月開催している Smalltalk勉強会 で、F-Script の概要紹介といくつかのデモからなるプレゼンをさせて頂いた。タイトルは「F-Script 第2回 - Mac OS X が Smalltalk になっちゃう！」

このタイトルは、二年前の１月の Smalltalk勉強会 における 林徹也さんの F-Script に関する最初の発表のタイトルをそのまま拝借させて頂いて、その「第2回」としたものだ。


F-Script には 大別して二つの側面がある。

その一つは、「Cocoa オブジェクトスクリプティング環境の提供」という実践改善的な側面であり、もう一つは、「OOPAL（Object Oriented Programming and Array programming Language integration）という革新的言語環境の具体的提案」という実験的な側面だ。前回の林さんの発表資料には、その全体像が、偏り無く非常に簡潔に要約されている。

そこで、今回の発表では、この二つの側面のうち、前者に焦点を絞り、その特徴と使い方を、実例デモを含めてやや詳しく紹介することにした。

■ 最後に


本日の発表の現場では、出席者の皆さんから質問や関連情報などをたくさん戴き、大変有意義な時間になった。

今日の出席者の皆さん全員に、心から感謝したい。
